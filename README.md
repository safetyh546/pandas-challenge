# pandas-challenge
pandas homework

Analysis of Pymoli game purchases.  Data file is in the Resources folder, Purchase_data.csv.  Jupyter notebook is stored in the HeroesOfPymoli folder. 
Analysis covers Age Demographics of purchasers, overall count of purchases, count of players, sum of purchases, average purchaes, etc.
Word document "Pymoli Observable Trends" lists briefly some high level trends based on the analysis.
